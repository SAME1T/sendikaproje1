# Sendika İletişim Merkezi

Bu proje, sendika üyeleri için iletişim merkezi web uygulamasıdır.

## Özellikler

- Üyelik yönetimi
- Duyuru sistemi
- Şikayet bildirimi
- Talep oluşturma
- Randevu sistemi

## Kurulum

1. Projeyi klonlayın:
```bash
git clone [proje-url]
```

2. Bağımlılıkları yükleyin:
```bash
npm install
```

3. MongoDB'yi kurun ve çalıştırın

4. Uygulamayı başlatın:
```bash
npm run dev
```

## Teknolojiler

- Node.js
- Express.js
- MongoDB
- EJS Template Engine
- Bootstrap 5

## Geliştirme

Geliştirme modunda çalıştırmak için:
```bash
npm run dev
```
